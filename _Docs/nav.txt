	Навигация приложения (дерево действий)
	- устанавливается на уровне приложения в appmap.xml
	- типы элементов:
		- Catalog (раздел)
			.Access - открытый, для всех участников, для участников групп
			.Name
			.Title - для названия страницы
			.ShortTitle - для пути и заголовка браузера
			.IsHidden - не отображать в навигации
			.Tags
		- Item (действие)
			.Access - открытый, для всех участников, для участников групп
			.Name
			.Title - для названия страницы
			.ShortTitle - для пути и заголовка браузера
			.IsHidden - не отображать в навигации
			.Tags
		- Link (ссылка)
			.Access - открытый, для всех участников, для участников групп
			.Title - в карте сайта
			.Href
			.IsExternal
			.Tags

	Навигация сайта (дерево узлов)
	- устанавливается на уровне приложения в sitemap.xml
	- условия:
		[Name] --> узел
		[Slaves] --> группа
		--> ссылка		
	- типы элементов:
		- Catalog (группа)
			.Title - в карте сайта и в подчиненных
			.ShortTitle - для пути и заголовка браузера
			.IsHidden - не отображать в карте сайта, в подчиненных и в пути
			.Tags
		- Item (узел)
			.Name
			.Title - в карте сайта, в названии узла
			.ShortTitle - в пути, в заголовке браузера
			.IsHidden - не отображать в карте сайта
			.Tags
		- Link (ссылка)
			.Title - в карте сайта
			.Href
			.IsExternal
			.Tags
	

	Навигация узла (дерево страниц)
	- устанавливается в узловом _viewstart.cshtml
	- условия:
		[Name]
			[Slaves] --> каталог
			--> страница
		--> ссылка
	- типы элементов:
		- Catalog (каталог)
			.Name
			.Title
			.ShortTitle - для пути и заголовка браузера
			.IsHidden - не отображать в навигации, но отображать в пути как текст
			.HasDefaultStartPage
			.Tags
		- Item (страница)
			.Name
			.Title - для названия страницы
			.ShortTitle - для пути и заголовка браузера
			.IsHidden - не отображать в навигации
			.Tags
		- Link (ссылка)
			.Title
			.Href
			.IsExternal
			.Tags
